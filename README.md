# 🛡 NeuroFence

## AI-Powered LLM Security Scanner

NeuroFence is a cybersecurity research project that analyzes prompts given to Large Language Models (LLMs) to identify potential security risks.

## Objective

The aim of this project is to monitor AI model behavior and detect abnormal activities using activation analysis and anomaly detection techniques.

## Technologies

- Python
- PyQt6
- PyTorch
- Hugging Face Transformers
